# VibeCode Academy — Команда агентов

## Как использовать команду

Вызывай агентов через `@имя` или словами:

| Агент | Когда вызывать |
|---|---|
| `@ceo` | Стратегия, приоритеты, координация задач |
| `@analyst` | Аудит кода, что улучшить, технический долг |
| `@marketer` | Продвижение, каналы, позиционирование |
| `@designer` | UX/UI, онбординг, визуальные решения |
| `@senior-dev` | Написание кода, фичи, баги |
| `@devops` | Деплой, безопасность, инфраструктура |
| `@business-analyst` | Планы, roadmap, требования, метрики |
| `@teacher` | Контент уроков, квизы, план обучения |

## Цепочки (когда задача требует нескольких агентов)

**Новая фича:**
`@business-analyst` (план) → `@designer` (UX) → `@senior-dev` (реализация) → `@devops` (деплой)

**Аудит проекта:**
`@analyst` (находит проблемы) → `@ceo` (приоритизирует) → нужный агент (исправляет)

**Запуск продукта:**
`@analyst` (готовность) → `@marketer` (стратегия) → `@business-analyst` (план) → `@ceo` (go/no-go)

**Новый урок:**
`@teacher` (контент) → `@designer` (визуал) → `@senior-dev` (добавить в БД)
